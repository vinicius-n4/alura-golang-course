# alura-store

## Run Locally

Go to the project directory.
```bash
$ cd alura-golang-course/alura-store
```

Run containerized PostgreSQL with the following command.
```bash
$ docker compose up -d
```

Start the server.
```bash
$ go run main.go
```

Open the browser and access `http://localhost:8000` to interact with the interface.
